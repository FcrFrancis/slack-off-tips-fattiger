# slack-off-tips-fattiger

胖虎哥摸鱼办公室文案生成器

## 示例

```
【胖虎哥摸鱼办公室】9月13日
上午好，摸鱼人，工作再累，一定不要忘记摸鱼哦！
有事没事起身去茶水间去厕所去廊道走走，别老在工位上坐着，钱是老板的，但命是自己的
距离【本周周末】还有5天
距离【中秋假期】还有5天
距离【国庆假期】还有17天
距离【跨年假期】还有109天
距离【春节假期】还有140天
```

## 指令

```bash
# 安装
npm i -g https://github.com/FcrFrancis/slack-off-tips-fattiger

# 显示今天的摸鱼文案
moyu show
# 增加自定义节日锚点
moyu mark 2021-12-25 圣诞节`
```
